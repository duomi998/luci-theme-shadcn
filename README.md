<h1 align="center">Shadcn LuCI Theme</h1>
<p align="center"><strong>A modern sidebar LuCI theme for OpenWrt, built with shadcn/ui design language.</strong></p>

## About

**luci-theme-shadcn** reskins the OpenWrt LuCI admin UI with a contemporary dashboard feel: collapsible sidebar navigation, semantic design tokens, and light/dark mode.
